# Project: N-Body Simulator

## Vision
A high-performance, browser-based N-body simulation utilizing the Barnes-Hut algorithm to simulate thousands of particles with gravitational interactions.

## Context
- **Primary Goal**: Scalability and performance.
- **Target Platform**: Web (WASM + Canvas).
- **Core Technology**: Rust for physics, JS for UI/Rendering.

## Focus Areas
1. **Simulation Fidelity**: Efficient Barnes-Hut implementation.
2. **Rendering Performance**: Optimized Canvas 2D drawing.
3. **User Control**: Interactive parameters (G constant, softening, particle count).
